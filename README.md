## Tech Stack

- **Client:** React JS
- **Server:** Node JS
- **Database:** MongoDB

# Run server locally

Clone the project

```bash
  git clone https://github.com/PranavNair01/Lohit_Kriti_Software_Dev_24
```

Go to the project directory

```bash
  cd jeeva_ai
```

Go to the backend directory

```bash
  cd backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Go to the frontend directory

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
